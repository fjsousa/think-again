#4.11 The Conditional

if P, then Q follows -(P&-Q)

and

-(P&-Q) follows from if P, then Q

###Modus Ponens (form of inference)
From the premises:

- P
- if P, then Q

Conclusion:

- Q

###Modus Tollens 
From the premises:

- Q
- if P, then Q

Conclusion:

- P

If the argument:

- P
- C

is valid, then the conditional 

If P, then C is true
